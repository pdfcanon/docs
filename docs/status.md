# Status

For real-time service status, incident history, and uptime metrics, visit the PDFCanon status page:

**[status.pdfcanon.com](https://status.pdfcanon.com)**

## Subscribe to status updates

You can subscribe to email or webhook notifications for status changes at [status.pdfcanon.com](https://status.pdfcanon.com).

| Portal | `app.pdfcanon.com` — web dashboard |
